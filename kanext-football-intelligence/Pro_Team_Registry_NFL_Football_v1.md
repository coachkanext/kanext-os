# PRO TEAM REGISTRY -- NFL (2026) v1

## Purpose
Maps every NFL team's offensive/defensive system, roster context, and team window. Enables system fit % and draft-range-appropriate evaluation for any prospect against any NFL team.

## System Taxonomy (from Football File 02)
**Offense (8):** Spread, Air Raid, RPO, Pro Style, West Coast, Power Run, Option/Triple Option, Pistol
**Defense (6):** 4-3 (Over/Under), 3-4 (Odd Front), Nickel/4-2-5, 3-3-5 (Wide Tackle Six), 4-4 (Eight-Man Front), 46 Defense

Reference Cap: 2026 NFL Salary Cap = $301.2M

## How Team Window Affects Draft Priority

| Window | Draft Position | What They Draft For |
|--------|---------------|-------------------|
| Rebuilding | Top 10 (#1-10) | PEAK CEILING. Best player available. Fit/overlap secondary — roster will change around this pick. |
| Rising | Mid-first (#11-20) | 3-YEAR PROJECTION + FIT. Have young talent, need complementary pieces. |
| Contending | Late first (#21-32) | MEDIAN OUTCOME + FIT. Need contributors now. High floor, system-ready. |
| Retooling | Varies | Depends on pick position. If top-10, lean peak. If late, lean fit. |

## HOW TO USE FOR DRAFT / SYSTEM FIT

1. Identify the prospect's archetype from Mode 1 eval
2. Look up the drafting team's offensive and defensive system + window + draft priority
3. If team drafts for PEAK — rank prospects by Peak Ceiling KR, de-emphasize fit
4. If team drafts for 3-YEAR — rank by 3-Year Projection, weight system fit 30-40%
5. If team drafts for MEDIAN — rank by Median Outcome, weight system fit 50%+
6. Run archetype-vs-system interaction from File 04 for fit %
7. Factor roster overlap (does this player's position conflict with existing core?)
8. Factor cap implications (can the team afford this player's second contract?)

## SCHEME IDENTIFICATION NOTES

Offensive system identification is based on public coaching analysis, tendencies, and personnel usage. Most NFL teams run hybrid offenses that blend 2-3 systems. The PRIMARY system listed is the base identity — the scheme that defines the team's offensive DNA. Secondary systems are listed where the blend is significant enough to affect draft/system fit evaluation.

Defensive system identification follows the same logic. Most teams have a base front but adjust to sub-packages (nickel, dime) on 60%+ of snaps. The listed defense is the base identity.
