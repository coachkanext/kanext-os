# 1. Development Intelligence Engine

## 0. Purpose
The Development Intelligence Engine takes everything the system knows about an athlete and translates it into actionable intelligence for skill progression, role development, and career planning.

It answers five questions for any evaluated athlete:
1. **Where are you now?** - Truth summary across every level and role
2. **Where should you be?** - Best-fit targets ranked by actual team impact
3. **What are you worth there?** - Value at each target (scholarship/role implication)
4. **What's the gap?** - Specific skills holding you back, with exact progressions needed
5. **What's the path?** - Prioritized development roadmap with projected timeline

This engine does NOT evaluate athletes. It does NOT modify Athlete KR, Team KR, archetypes, or any upstream output. It reads governed truth and produces downstream recommendations only.

## 1. Consumers
- The athlete: "What should I work on? What level should I compete at? Am I ready to fly at the college level?"
- The athlete's current coach: "How do I develop this athlete? What's their ceiling? What skills should we prioritize?"
- Recruiting coordinators: "Does this athlete fit our roster needs? What role can they fill? How quickly can they contribute?"
- All-Star coaches: "Can this athlete step up to Level 6? What skills do they need first?"
- Parents: "What level is my child at? What's a realistic path to a college roster?"

## 2. Skill Progression Framework

### Purpose
Cheer skills follow a strict progression hierarchy. You cannot safely learn a standing full without first mastering a standing layout. You cannot safely fly at extension without first mastering prep-level stunts. The Development Engine enforces this progression.

### Tumbling Progression Ladder

#### Standing Tumbling
| Level | Skill | Prerequisite |
|-------|-------|-------------|
| L1 | Back walkover | Bridge kickover |
| L2 | Back handspring (spotted) | Back walkover + bridge snap-down |
| L3 | Back handspring (unspotted) | 10 consecutive spotted BHS with good form |
| L4 | Standing back tuck | BHS series (3+) with strong snap + height |
| L5 | Standing layout | Back tuck with full extension at peak |
| L6 | Standing full (layout full twist) | Layout with head neutral and late twist initiation |
| L7 | Standing double full | Full with height to spare + clean landing |

#### Running Tumbling
| Level | Skill | Prerequisite |
|-------|-------|-------------|
| L1 | Round-off | Cartwheel with power |
| L2 | Round-off back handspring | Round-off with rebound + standing BHS |
| L3 | Round-off BHS series (3+) | Consistent RO-BHS with power through series |
| L4 | Running back tuck | BHS series with strong lift on final BHS |
| L5 | Running layout | Back tuck with full body extension |
| L6 | Running full | Layout with controlled twist |
| L7 | Running double full | Full with height to spare |
| L8 | Running double layout | Double full OR layout with extreme height |

#### Connected Tumbling
| Level | Skill | Prerequisite |
|-------|-------|-------------|
| L1 | BHS-BHS-back tuck | BHS series + standing tuck |
| L2 | BHS-layout | BHS series + running layout |
| L3 | BHS-full | Running full + strong BHS series |
| L4 | Whip-back tuck | Running layout + back tuck |
| L5 | Whip-full | Whip-back tuck + running full |
| L6 | Full-layout | Running full + strong rebound |
| L7 | Whip-double | Whip-full + running double |

### Stunt Progression Ladder (Flyer)

| Level | Skill | Prerequisite |
|-------|-------|-------------|
| L1 | Thigh stand | Basic balance + trust with bases |
| L2 | Prep (waist level) liberty | Thigh stand stability + single-leg balance |
| L3 | Extension (overhead) two-leg | Prep-level consistency + bases at extension capability |
| L4 | Extension liberty | Extension two-leg stability + strong core |
| L5 | Body positions at extension (heel stretch, arabesque) | Extension liberty hold 3+ seconds |
| L6 | Advanced positions (scorpion, scale) | Heel stretch + flexibility |
| L7 | Elite positions (needle, bow and arrow) | Scorpion + elite flexibility |
| L8 | Transitions at extension (tic-tock, switch-up) | Stable in multiple positions + strong bases |
| L9 | Full-up to position | Tic-tock + trust + timing with bases |

### Stunt Progression Ladder (Base)

| Level | Skill | Prerequisite |
|-------|-------|-------------|
| L1 | Thigh stand support | Basic lifting technique + grip |
| L2 | Prep-level stunt (two bases) | Thigh stand + overhead press capability |
| L3 | Extension stunt (two bases) | Prep-level consistency + overhead strength |
| L4 | Liberty support at extension | Extension + timing with flyer balance |
| L5 | Toss to hands | Extension grip + explosive power |
| L6 | One-arm at prep level | Two-arm extension + wrist/grip strength |
| L7 | One-arm at extension | One-arm prep + elite strength/balance |
| L8 | Full-up support (catching twist) | Toss timing + catch technique |

### Dismount Progression
| Level | Skill | Prerequisite |
|-------|-------|-------------|
| L1 | Step-off / pike down | Basic trust |
| L2 | Cradle | Step-off + catch technique |
| L3 | Twist cradle | Cradle + timing |
| L4 | Kick full | Twist cradle + toss height |
| L5 | Kick double | Kick full + elite toss height + catch precision |

### SAFETY RULE
No athlete may be assigned a skill that is more than ONE level above their current demonstrated level. A skill two or more levels above the current demonstrated level requires progressive mastery through each intermediate level. This is a HARD RULE that cannot be overridden by coach request.

## 3. Development Timeline Estimation

### Purpose
Estimate how long it takes an athlete to gain a new skill level based on their current KR, physical tools, and training environment.

### Timeline Factors
| Factor | Impact on Timeline |
|--------|-------------------|
| Athlete AKR (physical tools) | Higher AKR = faster progression. AKR 85+ reduces timeline by 25%. AKR below 70 extends by 25%. |
| Athlete IQKR (coachability) | Higher IQKR(coachability) = faster learning. IQKR 85+ reduces by 15%. Below 70 extends by 15%. |
| Training frequency | 5+ days/week: baseline. 3-4 days: +25% timeline. 1-2 days: +50% timeline. |
| Facility access | Spring floor + pit: baseline. Spring floor only: +10%. Hard floor only: +30% (and safety-limited ceiling). |
| Coaching quality | Nationally recognized program: -15%. Standard coaching: baseline. Volunteer/limited coaching: +20%. |
| Age | Under 14: fastest learning window. 14-18: standard. 18+: +10-20% per skill level (adult learners need more time). |
| Injury history | Relevant prior injury: +15-25% for related skills (e.g., ankle injury extends tumbling timeline). |

### Standard Timeline Estimates (per skill level gain)

#### Tumbling (assuming 4-5 days/week training, spring floor access)
| Current to Next | Typical Timeline |
|----------------|-----------------|
| BHS to Back Tuck | 2-4 months |
| Back Tuck to Layout | 1-3 months |
| Layout to Full | 3-6 months |
| Full to Double Full | 6-12 months |
| Each connected skill level | 2-4 months after component skills are mastered |

#### Stunting (assuming 4-5 days/week training, proper equipment)
| Current to Next | Typical Timeline |
|----------------|-----------------|
| Prep to Extension | 1-2 months |
| Extension to Liberty | 2-4 months |
| Liberty to Advanced Positions | 3-6 months (flexibility dependent) |
| Standard to Transition Skills | 4-8 months |
| Transitions to Full-Up | 6-12 months |

### Development KR Projection
Given an athlete's current KR and development plan:
- **1-Year Projection:** Current_KR + (estimated skill gains x trait weight improvement)
- **2-Year Projection:** 1-Year + additional gains (diminishing returns after Year 1)
- Maximum single-year KR improvement: +8 (beyond this is unrealistic without exceptional circumstances)
- Maximum two-year KR improvement: +12

Development projections carry confidence based on:
- Data quality of current evaluation: 60-80% of projection confidence
- Training environment quality: modifies by +/- 10%
- Athlete age and development history: modifies by +/- 10%

# 2. Transition Engine

## Purpose
Model career transitions for athletes moving between competitive contexts:
- High school to college
- All-Star to college
- College to All-Star (post-college)
- College to professional (performance companies, coaching)
- Level-to-level within All-Star (L4 to L5, L5 to L6, etc.)

## 2.1 High School / All-Star to College Transition

### Transition Assessment
For each prospective college program:
1. **KR Translation:** Read the athlete's KR against the college-level legend
2. **Role Fit:** Does the college team need this athlete's role?
3. **Skill Readiness:** Are the athlete's skills at or near the college level's expectation? Use the Skills Inventory and compare against the college legend's tier descriptions.
4. **Development Gap:** What skills must the athlete gain to contribute at the target level? Use the Skill Progression Framework to estimate timeline.
5. **Physical Readiness:** Does the athlete's AKR suggest they can handle the physical demands of college competition?

### College Fit Score
College_Fit = (KR_Translation_Score x 0.40) + (Role_Need x 0.25) + (Skill_Readiness x 0.20) + (Development_Potential x 0.15)

Where:
- KR_Translation_Score = how the athlete's KR reads at the college level (0-100)
- Role_Need = how much the college team needs this athlete's role (0-100, where 100 = critical need)
- Skill_Readiness = percentage of target-level skills the athlete currently possesses
- Development_Potential = projected KR gain in 1-2 years based on tools and trajectory

### Output
For each target program:
- Fit Score
- Role projection (what role would this athlete play on this team?)
- Contribution timeline (contribute immediately? Redshirt year? 2-year project?)
- Scholarship likelihood (based on fit score and program scholarship budget)
- Development plan needed (specific skills to gain before or during first year)

## 2.2 Level-to-Level All-Star Transition

### Assessment
For athletes moving from one All-Star level to the next:
1. Compare current Skills Inventory against the higher level's skill expectations
2. Identify gaps (skills required at the new level that the athlete doesn't have)
3. Estimate timeline to close each gap
4. Compute expected KR at the new level (same KR, different legend read)
5. Assess whether the athlete will be competitive at the new level immediately or needs development time

### Transition Readiness Score
Transition_Ready = (Skills_Covered / Skills_Required) x 100

- 90%+ coverage: Ready. Can transition immediately and contribute.
- 75-89% coverage: Nearly ready. 2-4 months of skill development recommended.
- 60-74% coverage: Developing. 4-8 months needed. May compete at the new level but with limitations.
- Below 60%: Not ready. Premature transition risks safety and confidence.

## 2.3 Post-College / Professional Transition

### Career Pathways
1. **All-Star Coaching:** Requires IQKR(leadership) 80+, experience across multiple levels, strong communication skills. Credential: USASF coaching certification.
2. **College Coaching:** Requires competitive experience at college level, IQKR 80+, recruiting network. Path: volunteer assistant to assistant to head coach.
3. **Professional Performance:** Cirque du Soleil, professional dance companies, NBA/NFL sideline. Requires elite AKR, JKR, performance quality. Highly selective - KR 90+ recommended.
4. **All-Star Athlete (Worlds-level):** Post-college athletes competing on elite All-Star teams. Requires maintained skill level and AKR. Age considerations (peak physical years typically 18-25 for cheer).

### Post-College KR Maintenance Model
Athletes who stop training full-time will see KR decline:
- Year 1 post-college: -2 to -5 KR (skill maintenance without daily practice)
- Year 2 post-college: -3 to -7 additional KR decline
- Exception: Athletes who join All-Star programs and maintain training: -0 to -2 per year (maintained)
- Exception: Athletes with elite AKR (90+) decline slower (-1 to -3 per year)

# 3. Injury Risk Intelligence

## Purpose
Model injury risk based on athlete profile, training load, skills performed, and role assignment. This is a SAFETY system that supplements the Safety Protocol defined in the SKILL file.

## 3.1 Common Cheer Injuries by Role

### Flyer Injuries
| Injury | Risk Factors | Prevention Signal |
|--------|-------------|-------------------|
| Ankle sprains/fractures (landing) | Poor dismount technique, inadequate catch | Low SKR(dismount), low Base SKR(catch) |
| Wrist injuries (cradle impact) | Improper catch position | Low catch consistency in stunt group |
| Concussion (falls from height) | Stunt instability, inadequate spotting | Low SKR(flyer stability), low BackSpot KR |
| Lower back strain | Hyperextension in body positions | Flexibility without adequate core strength |
| Neck injuries (inverted skills) | Pyramid inversions, braced flips | Advanced skills without progressive mastery |

### Base Injuries
| Injury | Risk Factors | Prevention Signal |
|--------|-------------|-------------------|
| Shoulder injuries (overhead support) | Excessive flyer weight, poor technique | Low AKR(strength) relative to flyer weight |
| Wrist/hand injuries (grip) | Grip breaks, catching falls | Low SKR(catch consistency), fatigue |
| Lower back injuries (lifting) | Improper lifting mechanics | Poor technique score in base assessment |
| Knee injuries (squat position) | Deep squat under load, lateral movement | Low AKR with high stunt demands |

### Tumbler Injuries
| Injury | Risk Factors | Prevention Signal |
|--------|-------------|-------------------|
| Ankle sprains (landing) | Hard landings, over-rotation | Low TKR(execution), low AKR(body control) |
| ACL/knee injuries | Twisting on landing, fatigue | History of knee issues, low endurance |
| Wrist fractures (tumbling on hands) | Under-rotation, wrist collapse | Low TKR(power) for skill attempted |
| Concussion (under-rotation) | Attempting skills above ability | SAFETY FLAG: skill above demonstrated level |
| Lower back stress fractures | Repetitive hyperextension | High training volume, low core strength |

## 3.2 Injury Risk Score
For each athlete, compute an Injury Risk Score (1-10, where 10 is highest risk):

Risk_Score = Base_Risk + Role_Risk + Training_Risk + History_Risk

**Base_Risk (1-3):**
- AKR 85+: 1
- AKR 70-84: 2
- AKR below 70: 3

**Role_Risk (0-3):**
- Flyer performing advanced skills: +2
- Base supporting at near-capacity: +2
- Tumbler performing at highest level: +1
- Back spot: +0 (lowest injury role)
- Skills above demonstrated level: +3 (AUTOMATIC RED FLAG)

**Training_Risk (0-2):**
- Proper facility, adequate coaching, managed load: +0
- Limited facility or high training volume: +1
- No spring floor, high volume, limited coaching: +2

**History_Risk (0-2):**
- No injury history: +0
- Minor injury history: +1
- Recurring or major injury history: +2

### Risk Interpretation
- Risk Score 1-3: GREEN. Standard training and competition. Monitor normally.
- Risk Score 4-6: YELLOW. Increased monitoring. Consider workload management. Ensure athletic trainer access.
- Risk Score 7-8: ORANGE. Active risk management required. Limit high-risk skills. Mandatory athletic trainer oversight.
- Risk Score 9-10: RED. Safety concern. Restrict to demonstrated skills only. Medical evaluation recommended.

## 3.3 Training Load Monitoring
For programs with practice data:
- Track weekly training hours and skill repetitions
- Flag sudden increases in training load (>15% week-over-week increase)
- Flag athletes with high training load AND low endurance KR
- Recommend rest/recovery when training load exceeds sustainable thresholds
- Model the relationship between training load and competition performance (over-training decreases performance)

# 4. Coaching Impact Modifier

## Purpose
Quantify the development impact of different coaching environments. Used in Development Intelligence to adjust skill progression timelines and KR projections.

## 4.1 Coaching Tier Classification

### Tier 1: Elite Program (Modifier: 1.20)
- Nationally recognized coaching staff
- Full-time coaching positions
- Spring floor, proper mats, pit access
- Athletic training staff on-site
- History of producing college athletes and/or All-Star Worlds competitors
- Systematic skill progression program documented
- 3+ coaches on staff (head coach + specialists)

### Tier 2: Strong Program (Modifier: 1.10)
- Experienced coaching staff with competitive records
- Dedicated practice facility (may not have pit)
- Athletic trainer access (may not be on-site)
- Track record of athlete development
- 2+ coaches on staff

### Tier 3: Standard Program (Modifier: 1.00)
- Competent coaching staff
- Adequate facility
- Standard training schedule
- Some athlete development track record
- 1-2 coaches

### Tier 4: Limited Program (Modifier: 0.90)
- Volunteer or part-time coaching
- Limited facility (gym mats on hard floor, shared space)
- No dedicated athletic training
- Limited competitive experience on staff
- 1 coach

### Tier 5: Minimal Program (Modifier: 0.80)
- No formal coaching structure
- No proper facility
- Self-taught or peer-taught skills
- Safety concerns due to lack of supervision

## 4.2 Application
The Coaching Impact Modifier adjusts development timelines:
Adjusted_Timeline = Base_Timeline / Coaching_Modifier

Example: A skill that typically takes 4 months to develop:
- Tier 1 program: 4 / 1.20 = 3.3 months
- Tier 3 program: 4 / 1.00 = 4.0 months
- Tier 5 program: 4 / 0.80 = 5.0 months

## 4.3 Facility Impact on Skill Ceiling
Facilities create hard ceilings on certain skill development:

| Facility | Skill Ceiling Impact |
|----------|---------------------|
| Spring floor + pit + spotting rig | No ceiling imposed by facility |
| Spring floor + pit (no rig) | Standing tumbling L7, running L8 achievable |
| Spring floor only (no pit) | Standing L6, running L7 achievable with elite coaching |
| Dead floor (no spring) | Standing L4, running L5 maximum safe progression |
| Grass/outdoor | Standing L3, running L4 maximum. NOT recommended for serious development. |

These are safety-imposed ceilings. The system will not project skill development beyond what the facility safely supports, regardless of athlete talent.

# 5. Recruiting Intelligence

## Purpose
Help coaches identify and evaluate prospective athletes for their program.

## 5.1 Recruiting Needs Assessment
Based on current roster analysis:
1. Identify roles that will need replacement (graduating seniors, projected departures)
2. Identify roles where current depth is thin
3. Identify skills that the team cannot currently perform due to roster limitations
4. Compute the KR improvement needed to reach competition goals
5. Prioritize recruiting targets by Expected Team KR Impact (from File 03 Roster Decision Intelligence)

## 5.2 Recruit Evaluation
For each prospective recruit:
1. Run standard evaluation (File 01) with available data
2. Compute Level Tier Map (how does this athlete project at our level?)
3. Assess Role Fit against roster needs
4. Compute Development Projection (what will this athlete look like in 1-2 years with our coaching?)
5. Apply Coaching Impact Modifier to development timeline
6. Compute Scholarship Value relative to PTV

## 5.3 Recruiting Watchlist
Maintain a ranked list of recruiting targets:
- Rank by Expected Team KR Impact
- Flag athletes whose recruitment timeline aligns with roster needs
- Flag athletes with competing offers (if known)
- Update rankings as new competition data becomes available

## 5.4 Transfer Portal Intelligence (College)
For athletes entering the transfer portal:
- Run evaluation with available data from prior institution
- Apply Role Mismatch consideration (athlete may have been suppressed in prior role)
- Compute immediate impact (no development timeline - portal athletes need to contribute now)
- Flag chemistry risk (new partner groups, new coaching system)
- Compare against recruiting class alternatives (is a portal athlete a better use of scholarship than a high school recruit?)
