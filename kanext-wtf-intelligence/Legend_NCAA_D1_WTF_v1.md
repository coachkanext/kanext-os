# Legend - NCAA D1 Women's Track and Field v1

---

## SCOPE
This legend file provides KR tier interpretation for NCAA Division I women's track and field. It covers all D1 Major Classes (Power 4, Mid-Major, Low-Major) with separate tier labels where appropriate.

**ALL BENCHMARKS ARE WOMEN'S-SPECIFIC.** Women's implements, hurdle heights, and performance standards.

---

## HOW TO USE THIS LEGEND
- This file INTERPRETS KR. It does not produce KR.
- An athlete's KR is ONE universal number. This legend shows what that number MEANS at the D1 level.
- For D1, tiers differ by Major Class. A KR of 88 means different things at a Power 4 program vs a Low-Major.

## SCHOLARSHIP CONTEXT (D1)
- NCAA D1 women's track and field: 18 equivalency scholarships
- Can be split across multiple athletes
- Typical roster: 30-80 athletes (many partial scholarships and walk-ons)
- Scholarship value varies by program ($5K-$60K+ per full scholarship depending on institution)

## GOVERNANCE
- This legend is display-only. It interprets KR; it does not produce KR.
- One athlete = one KR = one interpretation per level.
- Major Class (P4/MM/LM) determines which tier table to use within D1.
- All benchmarks are women's-specific. All implement weights are women's.
