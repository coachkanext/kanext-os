# ACQUISITION SCOUTING AND DEAL OPS
## v1.0

---

## 0. Scope

This is the single authoritative document for acquisition target sourcing, pipeline management, and deal execution operations. It replaces no prior document (first version).

Scouting and Deal Ops is the operational playbook for finding, qualifying, approaching, negotiating, and closing acquisitions. It is the acquisition equivalent of basketball's Game Ops (pregame scout, in-game ops, halftime adjustments, postgame analysis) applied to the M&A process.

This document does not evaluate targets (that is File 01/02), does not simulate deals (that is File 04), and does not manage post-close operations (that is File 06). It governs the process from "we need to acquire X" through "deal is closed."

## 1. Target Sourcing Engine

### 1.1 Purpose
Systematically identify acquisition targets that fit the KaNeXT portfolio thesis. Move from reactive ("someone brought us a deal") to proactive ("we identified the best targets in this category and are approaching them strategically").

### 1.2 Sourcing Channels

**Channel 1: Thesis-Driven Search**
- Start with the portfolio gap analysis (File 03, Section 6)
- Identify the target type needed (bank, VoIP, camera, fulfillment, enrollment, school, real estate)
- Define the ideal target profile using the Target Type Legend (File 02) as the specification
- Search systematically: industry databases, regulatory filings, trade association directories, broker networks

**Channel 2: Regulatory and Public Records**
- FDIC BankFind for bank targets (search by MDI designation, geography, asset size, CRA rating)
- FCC Universal Licensing System for VoIP/telecom targets
- NCES College Navigator for school targets (search by enrollment, geography, accreditation, programs)
- State Secretary of State filings for entity research
- County property records for real estate targets
- USPTO for technology/IP targets

**Channel 3: Broker and Advisor Networks**
- M&A brokers specializing in relevant sectors (education, banking, telecom, manufacturing)
- Investment banks with small to mid-market deal flow
- Accounting firms with clients considering succession or sale
- Law firms with clients in relevant sectors
- Industry-specific transaction advisors

**Channel 4: Direct Outreach**
- Identify targets through industry research and directly approach ownership
- Appropriate for targets not currently for sale but potentially interested
- Requires carefully crafted outreach (see Section 3: Approach Playbook)
- Higher conversion rate for mission-aligned targets (HBCU schools, MDI banks, faith-based institutions)

**Channel 5: Distressed and Opportunistic**
- FDIC failed bank list and assisted transaction opportunities
- Schools on accreditation warning or probation (NCES, accreditation agency public actions)
- Bankruptcy filings in relevant industries
- Regulatory enforcement actions creating forced sales
- Receiver and trustee sales

**Channel 6: Inbound**
- Targets or brokers approaching KaNeXT
- Referrals from advisors, board members, industry contacts
- Conference and event-generated leads

### 1.3 Target Qualification Screen

Before any target enters the evaluation pipeline (File 01), it passes through a rapid qualification screen. This takes 30 minutes maximum and uses only publicly available data. Purpose: filter out obvious non-fits before investing evaluation time.

**Qualification Criteria (all must pass):**

1. **Type Match.** Does this target fit one of the six defined target types? If it does not fit any category, it does not enter the pipeline.

2. **Size Match.** Is the likely transaction value within a feasible range given remaining fund capital? A $500M target when we have $100M remaining does not qualify.

3. **Geographic Relevance.** Is this target located in or serving a KaNeXT priority geography, OR is it a remote/digital operation where geography does not matter?

4. **Strategic Plausibility.** Can you articulate in one sentence why KaNeXT would acquire this target? If you cannot, it does not qualify.

5. **No Obvious Disqualifiers.** Active criminal investigation, debarment, sanctioned entity, terminal regulatory action, or severe reputational issue.

**Qualification Output:** PASS (enters evaluation pipeline) or FAIL (reason documented, target archived for potential future review).

### 1.4 Target Pipeline Registry

Every qualified target is tracked in the pipeline registry with the following fields:

| Field | Description |
|-------|------------|
| Target name | Legal entity name |
| Target type | From six defined types |
| Source channel | How we found this target |
| Qualification date | When it passed the screen |
| Pipeline stage | See Section 2 below |
| Owner | KaNeXT team member responsible |
| Priority | Critical / High / Medium / Low |
| Budget allocation | From Capital Deployment (if assigned) |
| Target KR | From evaluation (if completed) |
| Next action | What needs to happen next |
| Next action date | When it needs to happen |
| Notes | Free text for context |

## 2. Deal Pipeline Stages

Every acquisition moves through seven stages from identification to close. Each stage has defined entry criteria, activities, outputs, and exit criteria. No stage can be skipped.

### Stage 1: Identified
**Entry:** Target passed qualification screen.
**Activities:** Initial research. Public data gathering. Industry context. Preliminary assessment of strategic fit.
**Output:** Research summary (1-2 pages). Preliminary strategic rationale. Recommended priority level.
**Exit to Stage 2:** Decision to pursue (CEO approval for High/Critical priority targets).
**Timeline:** 1-2 weeks.

### Stage 2: Researched
**Entry:** Decision to pursue confirmed.
**Activities:** Deep public research. Run Data Gathering Protocol (File 07, SKILL.md). Build Target Profile (File 01). If target is approachable via broker or mutual contact, initiate preliminary conversation. If direct outreach needed, prepare approach materials.
**Output:** Target Profile populated to available depth. Preliminary Target KR estimate (Tier 3 or 4 confidence). Approach strategy defined.
**Exit to Stage 3:** Target Profile complete enough for outreach. Approach strategy approved.
**Timeline:** 2-4 weeks.

### Stage 3: Approached
**Entry:** Approach strategy approved. Outreach materials prepared.
**Activities:** Initial contact with target ownership or broker. Introduce KaNeXT. Gauge interest. If interested, establish NDA and begin information exchange.
**Output:** NDA executed (if proceeding). Seller/owner indication of interest. Initial price/structure expectations from seller. Information request list sent.
**Exit to Stage 4:** NDA signed, seller engaged, preliminary information received.
**Timeline:** 2-6 weeks (varies widely by seller responsiveness).

### Stage 4: Under Evaluation
**Entry:** NDA signed. Proprietary information being received.
**Activities:** Full Target Evaluation (File 01, full pipeline). Component KR scoring with proprietary data. System Fit assessment. Integration complexity assessment (Mode 4). Risk assessment (Mode 5). Preliminary valuation modeling (Mode 6).
**Output:** Complete Target Evaluation with Tier 2 or Tier 1 confidence. Valuation range. Deal structure recommendation. Integration plan (preliminary). Risk flags with mitigation plans.
**Exit to Stage 5:** Evaluation complete. Decision to proceed with LOI (CEO approval).
**Timeline:** 4-8 weeks (depends on data availability and complexity).

### Stage 5: LOI / Term Sheet
**Entry:** Evaluation complete and affirmative. Decision to proceed.
**Activities:** Draft LOI or term sheet. Negotiate key terms (price, structure, exclusivity, contingencies, timeline). Run Deal Simulation (File 04) with proposed terms. Run stress tests on proposed deal. Confirm capital availability.
**Output:** Executed LOI or term sheet. Exclusivity period established. Due diligence scope defined. Timeline to close agreed.
**Exit to Stage 6:** LOI/term sheet executed by both parties.
**Timeline:** 2-4 weeks of negotiation.

### Stage 6: Due Diligence
**Entry:** LOI/term sheet executed. Exclusivity granted.
**Activities:** Execute Due Diligence Protocol (File 07, Mode 7). Financial DD, legal DD, operational DD, regulatory DD, site visit. Update Target Evaluation with DD findings. Re-run valuation with verified data. Finalize deal structure. Negotiate definitive agreement. Prepare integration plan (detailed).
**Output:** DD completion checklist (all items addressed). Updated Target KR with Tier 1 confidence. Definitive agreement negotiated. Integration plan finalized. Regulatory applications filed (if required). Board/governance approvals obtained (if required).
**Exit to Stage 7:** DD complete. No material adverse findings (or findings addressed in deal terms). Definitive agreement ready for execution. All approvals obtained or timeline confirmed.
**Timeline:** 4-12 weeks (regulated targets: 6-18 months including regulatory approval).

### Stage 7: Close
**Entry:** Definitive agreement ready. All conditions satisfied or waived.
**Activities:** Execute definitive agreement. Fund capital transfer. Title transfer (real estate). License transfer (regulated). Day 1 communications (see File 06, Integration Tracking). Transition to post-close operations.
**Output:** Deal closed. Capital deployed. Ownership transferred. Integration Phase 0 complete (File 06).
**Exit:** Target moves from deal pipeline to post-acquisition monitoring (File 06).
**Timeline:** 1-2 weeks for execution mechanics.

### Pipeline Stage Summary

| Stage | Name | Duration | Key Decision |
|-------|------|----------|-------------|
| 1 | Identified | 1-2 weeks | Pursue or archive |
| 2 | Researched | 2-4 weeks | Approach or hold |
| 3 | Approached | 2-6 weeks | Engage or withdraw |
| 4 | Under Evaluation | 4-8 weeks | LOI or pass |
| 5 | LOI / Term Sheet | 2-4 weeks | Proceed to DD or walk |
| 6 | Due Diligence | 4-18 weeks | Close or walk |
| 7 | Close | 1-2 weeks | Execute |

**Total timeline (non-regulated):** 15-38 weeks (4-9 months)
**Total timeline (regulated):** 25-52+ weeks (6-13+ months)

## 3. Approach Playbook

### 3.1 Purpose
Define how KaNeXT approaches potential acquisition targets. The approach sets the tone for the entire relationship. A well-crafted approach opens doors. A clumsy approach closes them permanently.

### 3.2 Approach by Seller Type

**Approach A: Mission-Aligned Institution (HBCU, faith-based school, MDI bank, community institution)**
- Lead with mission. KaNeXT's institutional thesis resonates with mission-driven sellers.
- Opening message: Who we are (institutional OS company, HBCU operator, Christian institutional identity), what we are building (network of institutions operating at the highest level), why their institution matters to the mission.
- Do NOT lead with price. Do NOT lead with "we want to acquire you." Lead with partnership language.
- Initial ask: Exploratory conversation about shared mission and potential collaboration.
- Tone: Respectful, mission-first, long-term oriented. These are legacy institutions. Their boards and stakeholders care about continuity and mission preservation.

**Approach B: Operating Company (VoIP, camera, fulfillment, enrollment, technology)**
- Lead with strategic fit and growth opportunity.
- Opening message: What KaNeXT is building, why their company is uniquely positioned to be part of it, what the acquisition would mean for their business (not just for KaNeXT).
- Acknowledge the seller's achievement. They built something. Recognize it.
- Present the value proposition for the seller: access to 1,600+ institutional customers, integration into a growing platform, growth capital, operational support.
- If seller is a founder: address legacy concerns. "Your company becomes the [VoIP/camera/fulfillment] layer for every institution in the network."
- Tone: Direct, professional, opportunity-focused.

**Approach C: Broker-Represented Target**
- The broker is the gatekeeper. Treat them professionally.
- Respond to broker marketing materials with specific, informed questions (shows you did homework, not tire-kicking).
- Request information package and NDA promptly. Speed matters with brokers - slow buyers get deprioritized.
- Clearly communicate: KaNeXT's capital availability, timeline expectations, decision-making authority (CEO makes the call, no committee delays).
- Tone: Efficient, credible, capitalized, decisive.

**Approach D: Distressed Target (FDIC-assisted, accreditation probation, bankruptcy)**
- Approach with sensitivity. Distressed sellers are under pressure. Predatory language or lowball posturing burns bridges.
- Lead with capability. "We have the operational infrastructure, capital, and institutional expertise to stabilize and grow this institution/company."
- For FDIC-assisted: work through the FDIC process. Submit expression of interest during the marketing period. The FDIC controls the timeline and process.
- For accreditation probation: approach the board, not the administration (board makes the decision). Present a credible operational plan that addresses the accreditation agency's concerns.
- Tone: Confident, capable, respectful of the situation.

**Approach E: Real Estate (Land Owner, Developer)**
- Lead with the development vision. Land sellers want to know what happens to their land.
- Present the campus vision: institutional development, community asset, long-term investment.
- For generational landowners: address legacy concerns. "This land becomes the campus for a major university and institutional hub."
- Engage local commercial real estate broker with market knowledge and seller relationships.
- Tone: Patient, vision-oriented, community-conscious. Real estate deals in established communities require community buy-in.

### 3.3 Approach Materials

Every approach includes:
1. **One-page KaNeXT overview** - What we are, what we are building, key facts (capital, institutions, technology)
2. **Target-specific rationale** - Why this specific target (not generic form letter)
3. **CEO introduction** - Brief bio establishing credibility and shared context where applicable
4. **Clear next step** - "We would welcome a 30-minute introductory conversation" (not "let us know if you are interested")

### 3.4 Approach Governance
- Every approach is reviewed by CEO before sending
- No approach promises price, structure, or terms before evaluation is complete
- No approach makes commitments KaNeXT cannot fulfill
- Every approach is documented in the pipeline registry with date, channel, content, and response

## 4. Deal Ops Confidence Gates

Like basketball's Scouting Confidence Gates (pregame and postgame), Deal Ops has confidence gates at each pipeline stage. These tell you how much you know and how much you are estimating.

### 4.1 Stage-Level Confidence

| Stage | Typical Confidence Range | Data Available |
|-------|------------------------|----------------|
| 1 - Identified | 20-35% | Public data only. Name, type, general industry. |
| 2 - Researched | 35-55% | Deep public research. Industry benchmarks. Preliminary financial estimates. |
| 3 - Approached | 40-60% | Public data plus seller's marketing materials or initial disclosures. |
| 4 - Under Evaluation | 60-85% | Proprietary financials. Management conversations. Operational data. |
| 5 - LOI | 70-85% | Evaluation complete. Key terms defined. Preliminary DD items addressed. |
| 6 - Due Diligence | 85-98% | Full DD. Audited financials. Legal review. Site visit. Management assessment. |
| 7 - Close | 95-100% | Everything verified. Conditions satisfied. Ready to execute. |

### 4.2 Minimum Confidence for Key Decisions

| Decision | Minimum Confidence |
|----------|--------------------|
| Proceed from Identified to Researched | 25% (just need enough to justify research time) |
| Proceed from Researched to Approached | 40% (must have strategic rationale supported by data) |
| Issue LOI | 65% (must have completed evaluation with at least Tier 3 data) |
| Begin Due Diligence | 70% (LOI terms must be grounded in real evaluation) |
| Close | 90% (DD must be substantially complete, all material items addressed) |
| Close with expedited DD | 80% (only permitted for deals under $10M with CEO written approval) |

### 4.3 Confidence Adjusters

Factors that increase confidence within a stage:
- Multiple independent data sources corroborate key facts (+5-10%)
- Management team transparent and responsive (+5%)
- Audited financials available (+10-15%)
- Site visit completed (+5-10%)
- Reference calls completed (customers, vendors, former employees) (+5%)
- Prior relationship with target or industry (+5%)

Factors that decrease confidence within a stage:
- Single source for critical data points (-5-10%)
- Management evasive or non-responsive (-10%)
- Financial data unaudited or management-prepared (-5-10%)
- No site visit possible (remote-only assessment) (-5%)
- Seller imposing unusual time pressure ("must close in 2 weeks") (-10%)
- Significant unexplained gaps in records (-10-15%)

## 5. Negotiation Intelligence

### 5.1 Purpose
Provide data-driven negotiation guidance based on evaluation outputs, market context, and deal dynamics.

### 5.2 Negotiation Position Inputs

Every negotiation position is informed by:
- Target KR and component KRs (what is the target actually worth based on quality)
- Valuation model output (Mode 6) - what should we pay
- Stress test results (File 04) - how much downside protection do we need
- Comparable transactions - what has the market paid for similar targets
- Seller motivation - why are they selling and how urgently
- Competition - are other buyers bidding
- Capital availability - what can we afford
- Strategic value - what is this worth to KaNeXT specifically (above standalone value)

### 5.3 Negotiation Guidance by Deal Dynamic

**Dynamic 1: Motivated Seller, No Competition**
- Position: Buyer advantage. Be patient. Let the seller come to you on price.
- Strategy: Offer below midpoint of valuation range. Include buyer-friendly terms (holdback, earnout, reps and warranties).
- Risk: Overplaying advantage and losing a good deal on a few percentage points.
- Principle: A fair deal that closes is better than a great deal that falls apart.

**Dynamic 2: Competitive Process (Multiple Bidders)**
- Position: Must differentiate on terms, not just price. KaNeXT's unique offer: institutional distribution, platform integration, mission alignment.
- Strategy: Bid at or slightly above midpoint. Differentiate on certainty of close (capitalized, no financing contingency), speed (CEO decision authority, no committee), and strategic value to the seller (their company becomes part of a national platform).
- Risk: Winner's curse (overpaying in a competitive auction).
- Principle: Set a walk-away price before entering the process. Do not exceed it in the heat of competition.

**Dynamic 3: Distressed Seller**
- Position: Buyer advantage but must balance with sensitivity and reputation.
- Strategy: Offer structure that is fair to the seller given their circumstances. Avoid predatory pricing that damages KaNeXT's reputation in the community. Focus on speed and certainty.
- Risk: Hidden liabilities in distressed targets. Regulatory complications. Community backlash.
- Principle: Treat distressed sellers with respect. The community watches how you treat people in difficulty.

**Dynamic 4: Mission-Aligned Partner (University, Church, Community Institution)**
- Position: Partnership, not adversarial negotiation. Both parties want the institution to thrive.
- Strategy: Structure that preserves mission and governance independence (IOA model). Philanthropic components where appropriate. Long-term commitment (25-year agreement, 100-year ground lease).
- Risk: Institutional politics. Board factions. Community opposition to change.
- Principle: This is not a transaction. It is a relationship. Structure accordingly.

**Dynamic 5: Regulatory Gatekeeper**
- Position: Regulated targets require regulator buy-in. The regulator is effectively a third party in the deal.
- Strategy: Pre-filing consultation with regulator. Address concerns proactively. Demonstrate capital adequacy, operational capability, and mission alignment. For MDI banks: emphasize minority ownership continuity. For schools: emphasize accreditation preservation and student welfare.
- Risk: Regulatory delay or denial.
- Principle: Never surprise a regulator. Engage early, be transparent, and address their concerns before they become conditions.

### 5.4 Walk-Away Triggers

The deal should be abandoned if any of the following are true:
- Purchase price exceeds the high end of valuation range AND no strategic premium justifies the difference
- DD reveals material undisclosed liabilities exceeding 15% of transaction value
- Key person departs during DD and no replacement is identified
- Regulatory pre-consultation indicates low probability of approval
- Seller demands terms that expose the fund to unquantifiable risk (no reps, no holdback, no indemnification)
- Stress test (File 04) shows the deal collapses under Scenario 1 (30% revenue miss) with no path to recovery
- Integration assessment (Mode 4) shows Transformational complexity (76-100) AND integration team capacity is already committed
- Seller behavior during DD suggests material integrity concerns (evasive, inconsistent statements, refusal to produce standard documents)

Walk-away is documented in the pipeline registry with reason and date. Target is archived, not deleted. Circumstances change and targets that are wrong today may be right in 2 years.

## 6. Deal Ops Playbook by Target Type

### 6.1 Bank Acquisition Playbook

**Pre-Approach:**
- Search FDIC BankFind for MDI-designated banks in target geography
- Review FDIC call reports for financial health (capital ratios, asset quality, earnings)
- Check CRA rating and BSA/AML compliance history
- Identify board composition and management team

**Approach:**
- Approach through mutual contact if possible (banking is a relationship industry)
- Lead with MDI preservation and community reinvestment commitment
- Emphasize deposit growth potential from KaNeXT institutional network

**Evaluation Specifics:**
- FKR weighted toward capital adequacy and asset quality (not revenue growth - banks grow differently)
- Regulatory Risk is the dominant RKR dimension
- System Fit heavily weighted on KayPay integration potential

**Structure:**
- Zero-upfront with deposit commitment is default
- Bank capitalization is an asset, not an expense
- Board composition must satisfy MDI requirements

**Regulatory Process:**
- FDIC change of control application
- OCC approval (if national charter)
- State banking regulator approval (if state charter)
- Timeline: 6-12 months minimum
- Pre-filing meeting with primary regulator before LOI

**Post-Close Priority:**
- Regulatory compliance continuity (no gap in BSA/AML, CRA reporting)
- IT system assessment and modernization plan
- Deposit growth activation (mandate school and KayPay pipeline)
- MDI and CDFI designation maintenance

### 6.2 School Acquisition Playbook

**Pre-Approach:**
- NCES College Navigator: enrollment, programs, accreditation, financial data
- Accreditation agency public actions: any warnings, probation, show-cause
- Board meeting minutes (if publicly available for state institutions)
- IRS Form 990 for nonprofit institutions (revenue, expenses, net assets, compensation)

**Approach:**
- IOA model (operational partnership) is preferred over outright acquisition
- Approach board chair or president through mutual contact
- Lead with institutional transformation vision, not financial terms
- Reference FMU as the model

**Evaluation Specifics:**
- Accreditation status is the single most critical factor. Without accreditation, there is no institution.
- Enrollment trajectory matters more than current enrollment level
- SKR heavily weighted on geographic fit with KaNeXT school network

**Structure:**
- IOA (Institutional Operating Agreement) is default
- Philanthropic transfer for campus real estate
- Operating agreement for technology, athletics, enrollment, campus operations
- University retains academic governance

**Regulatory Process:**
- Accreditation agency notification and approval (SACSCOC, HLC, WSCUC, etc.)
- State higher education board notification
- Title IV implications assessment
- HBCU designation preservation (if applicable)
- Timeline: 12-24 months

**Post-Close Priority:**
- Accreditation continuity (no disruption)
- KaNeXT OS deployment (Phase 1 per IOA)
- Enrollment infrastructure activation
- Athletics launch
- Faculty and staff retention

### 6.3 Infrastructure Company Playbook (VoIP, Camera, Fulfillment)

**Pre-Approach:**
- Industry research: market size, growth, key players, recent transactions
- Company research: revenue, team, technology, customer base
- Regulatory check: FCC licenses (VoIP), product certifications (camera), permits (fulfillment)

**Approach:**
- Direct outreach to founder or CEO
- Lead with growth opportunity: "Your company becomes the [communication/camera/fulfillment] layer for 1,600+ institutions"
- Present distribution advantage (KaNeXT mandate = guaranteed customer base)

**Evaluation Specifics:**
- FKR weighted toward recurring revenue and gross margin
- OKR weighted toward scalability (can this operation handle 10x volume)
- Technology stack quality is critical - will it integrate or need replacement

**Structure:**
- Cash acquisition is default for sub-$20M deals
- Earnout if valuation gap exists
- Equity rollover if founder retention critical
- Asset purchase if liability exposure elevated

**Regulatory Process:**
- VoIP: FCC notification, number porting coordination, CPNI compliance verification
- Camera: Product liability insurance, patent clearance
- Fulfillment: Warehouse permits, OSHA compliance, hazmat if applicable
- Timeline: 60-120 days (VoIP may add 30-60 days)

**Post-Close Priority:**
- Customer continuity (no service disruption)
- Key employee retention
- Technology assessment and integration roadmap
- Mandate school deployment plan
- KayPay integration

### 6.4 Real Estate Playbook

**Pre-Approach:**
- County property records: ownership, zoning, assessed value, liens, easements
- Environmental databases: EPA, state environmental agency
- Municipal planning: zoning map, comprehensive plan, future land use
- Comparable sales: recent transactions in area for similar-sized parcels

**Approach:**
- Engage local commercial real estate broker with seller relationship
- For large parcels: approach seller directly through mutual contact
- Lead with development vision (institutional campus, community asset)
- For generational landowners: emphasize legacy

**Evaluation Specifics:**
- Zoning and entitlements are the dominant FKR dimensions (not revenue - land does not generate revenue)
- Environmental condition is the dominant RKR dimension
- SKR heavily weighted on geographic fit and development potential

**Structure:**
- Cash purchase with standard contingencies (inspection, environmental, survey, zoning)
- Phased closing if multiple parcels
- Seller leaseback if seller needs transition time
- Philanthropic transfer if nonprofit land (FMU model)

**Process:**
- Title search and insurance
- Phase I (and Phase II if indicated) environmental assessment
- Survey
- Zoning confirmation or modification application
- Utility capacity assessment
- Traffic study (if required by municipality)
- Timeline: 60-120 days (longer if zoning requires modification)

**Post-Close Priority:**
- Development planning and design
- Entitlement and permitting process
- Construction procurement
- Community engagement

## GOVERNANCE RULES

1. **Pipeline discipline.** Every target enters through Stage 1 and exits through Stage 7 or archive. No shortcuts. No informal deals that skip stages.
2. **Documentation.** Every stage transition, every approach, every negotiation position, and every decision is documented in the pipeline registry.
3. **Confidence transparency.** Every pipeline stage has a confidence range. Decisions are never made at confidence levels below the minimum threshold for that decision type.
4. **Walk-away discipline.** Walk-away triggers are defined before engagement. They are not moved during negotiation heat. If a trigger fires, the deal is paused for reassessment at minimum.
5. **Reputation awareness.** KaNeXT's approach to targets reflects institutional values. Every interaction is professional, respectful, and honest. Bridges are not burned. Today's passed target may be tomorrow's partner.
6. **No truth mutation.** Deal Ops does not modify evaluations, simulations, or portfolio intelligence outputs. It consumes them. If negotiation produces new information, that information feeds back into a re-evaluation through the proper pipeline.
7. **CEO decision authority.** All LOI approvals, close decisions, and walk-away decisions are CEO decisions. This is not delegated. The pipeline provides data. The CEO makes the call.
