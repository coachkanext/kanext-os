# PRO TEAM REGISTRY -- NBA (2025-26)

## Purpose
Maps every NBA team's offensive and defensive system using the same 12 offense / 10 defense taxonomy from File 02. Enables system fit % calculation for any prospect against any NBA team.

## System Identification Status
v0 -- directional mapping based on publicly available coaching tendencies, scheme analysis, and play-type data. Will be refined with Synergy/Second Spectrum data when available.

## System Taxonomy (same as college, from File 02)
**Offense (12):** Spread Pick-and-Roll, 5-Out Motion, Read & React, Pace & Space, Dribble Drive, Princeton, Flex, Swing, Inside-Out, Moreyball, Heliocentric, Coach K
**Defense (10):** Containment Man, Pack Line, Pressure Man, Switch, No-Middle, Zone, Matchup Zone, Full-Court Press, Junk, Coach K

## HOW TO USE FOR DRAFT / SYSTEM FIT

1. Identify the prospect's archetype from Mode 1 eval
2. Look up the drafting team's offensive and defensive system
3. Run the archetype-vs-system interaction from File 04 (Interaction Library)
4. Produce system fit % for that prospect on that team

Example: AJ Dybantsa (Shot-Creating Wing) drafting to Memphis (Pace & Space / Containment Man)
- Offensive fit: Shot-creating wings thrive in Pace & Space (high fit)
- Defensive fit: Containment Man requires disciplined on-ball D (Dybantsa's DKR 76 = low fit)
- Overall system fit: ~78% -- offense fits, defense is the concern
