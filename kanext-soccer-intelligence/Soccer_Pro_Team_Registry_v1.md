# SOCCER PRO TEAM REGISTRY -- TOP 5 LEAGUES (2025-26) v1

## Purpose
Maps every club's offensive/defensive tactical system, manager, transfer budget tier, and squad window. Enables system fit % and transfer-target evaluation for any player against any club.

## System Taxonomy (from File 02)
**Offense (14):** Tiki-Taka, Gegenpressing, Counter-Attack, Possession, Direct/Long Ball, Wing Play, Inside Forward/Inverted Wing, False 9/Fluid Front, Target Man/Route One, Overlap/Underlap Width, Half-Space Exploitation, 3-2-5 Build-Up, Heliocentric, Manager K
**Defense (10):** High Press, Mid-Block, Low Block, Gegenpressing (Counter-Press), Man-Marking, Zonal Marking, Hybrid Press, Park the Bus, Offside Trap, Manager K

## Squad Window Definitions
| Window | Description | Transfer Priority |
|--------|-------------|------------------|
| Contending | Title/CL challenge. Squad is built. Fine-tuning. | System fit + immediate impact. High-floor signings. |
| Rising | Core established, building around it. 1-2 years from contending. | Development + fit. Medium-term upside. |
| Rebuilding | Major turnover needed. New project or manager. | Ceiling + potential. Accept short-term pain. |
| Developing | Youth-first model. Selling club. Pipeline to bigger leagues. | Youth with resale value. Low wages, high upside. |
| Retooling | Competitive but aging. Need to refresh without dropping off. | Balanced. Replace aging assets with ready contributors. |
