# PRO TEAM REGISTRY -- MLB (2026) v1

## Purpose
Maps every MLB team's offensive system, pitching philosophy, roster context, and competitive window. Enables system fit % and draft-range evaluation for any prospect against any MLB organization.

## System Taxonomy (from File 02)
**Offensive Systems (5):** Launch Angle/Power, Contact/Speed, Balanced, Small Ball, OBP-Focused
**Pitching Philosophies (5):** Strikeout-Dominant, Groundball-Dominant, Pitch-to-Contact, Bullpen-Heavy, Starter-Driven

## HOW TEAM WINDOW AFFECTS DRAFT PRIORITY

| Window | Draft Range | What They Draft For |
|--------|------------|-------------------|
| Contending | #20-40+ (late first, comp) | MEDIAN OUTCOME + FIT. Need contributors within 2 years. System-ready arms or polished college bats. |
| Rising | #8-20 | 3-YEAR PROJECTION. Have a young core, need complementary pieces. Development timeline matters. |
| Rebuilding | #1-10 | PEAK CEILING. Best talent available. HS arms and athletes with highest upside over 5-year window. |
| Retooling | Varies | Depends on slot. If top-10, lean peak. If later, lean fit + timeline. |

## PAYROLL TIERS (2026)

- **Low ($90-130M):** Twins, Pirates, Rays, Rockies, Angels, Nationals, Athletics, White Sox, Marlins

## HOW TO USE FOR DRAFT / SYSTEM FIT

1. Identify the prospect's archetype and component KRs from Mode 1/1P evaluation
2. Look up the drafting team's offensive system (for hitters) or pitching philosophy (for pitchers) + window + draft priority
3. Compute System Fit % against the team's demand profile
4. For PEAK picks: rank by Peak KR, weight system fit 10-20%
5. For 3-YEAR picks: rank by 3-Year Projection, weight system fit 30-40%
6. For MEDIAN picks: rank by Median Outcome, weight system fit 50%+
7. Factor in org development track record for pitchers (Tampa Bay, Cleveland, Milwaukee develop arms well; others less so)
8. Factor in park factor for hitters (Coors inflates, Oracle/T-Mobile suppress)

## BASEBALL-SPECIFIC DRAFT NOTES

Unlike basketball (where drafted players contribute immediately), baseball draft picks enter the minor league system. Key considerations:

- **Development org quality matters enormously.** Tampa Bay, Cleveland, Milwaukee, LA Dodgers consistently develop better than league average. Draft pick + strong dev org = higher probability of reaching Peak KR.
- **Park factor at MLB level affects hitter draft value.** A power hitter drafted by Colorado (Coors) vs Seattle (T-Mobile Park) has different offensive context even with identical talent.
- **Pitching philosophy alignment with draft pick's arsenal.** A groundball pitcher drafted by a strikeout-dominant org may be asked to change approach. System fit applies to pitchers too.
- **Minor league affiliate quality.** Some orgs have better coaching, facilities, and development infrastructure at the minor league level. This affects development timeline.
