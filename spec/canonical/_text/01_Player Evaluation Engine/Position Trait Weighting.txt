Position Trait Weighting (College)
Position Trait Weighting (College)
1. Purpose
Position Trait Weighting defines how evaluated attributes are valued differently by position in men’s college basketball.
This section translates raw trait scores into position-aware evaluation.
It answers one question only:
Given the same attributes, what matters more or less at this position?
2. Role of Position Weighting
Position weighting:
does not change trait scores
does not change archetype qualification
does not evaluate overall player quality
It applies relative importance to attributes after trait evaluation is complete.
3. Position Framework
Each position is defined by an Overall Position Framework (OPF).
The OPF specifies the relative importance of:
Offensive contribution
Defensive contribution
Physical and possession tools
The OPF does not:
assign roles
imply usage
dictate lineups
4. Offensive, Defensive, and Tools Components
For each position, attributes are grouped into:
Offensive KR (OKR)
Defensive KR (DKR)
Tools KR (TKR)
Rules:
OKR and DKR are visible and explicit
TKR exists but is intentionally hidden from surface outputs
All weights are fixed and position-specific
No dynamic adjustment occurs.
5. Position-Specific Weighting
Each position defines:
which attributes are emphasized
which attributes are de-emphasized
how offensive and defensive value is balanced
Examples:
Guards emphasize creation, decision-making, and perimeter defense
Wings emphasize shooting, containment, and team defense
Forwards emphasize finishing, physicality, and interior defense
Bigs emphasize rim protection, rebounding, and interior finishing
These are structural truths, not coaching preferences.
6. Determinism
Position Trait Weighting is:
deterministic
rule-based
repeatable
Given the same trait scores and position assignment, the same weighted result is produced.

7. Independence from Context
Position weighting does not change based on:
team system
opponent
role assignment
playing time
wins or losses
Context is handled downstream.
8. Downstream Use
Position-weighted outputs may be used by:
KR evaluation
player comparison within position
roster balance analysis
recruiting evaluation
Position weighting outputs may not be modified downstream.
9. Authority and Change Control
Position Trait Weighting is authoritative for:
position definitions
weighting structure
relative importance of attributes by position
Any change to:
weights
position definitions
included attributes
requires documentation, versioning, and approval.

Position Trait Weighting (Pro)
Position Trait Weighting (Pro)
1. Purpose
This section defines professional-level position trait weighting used for cross-level comparison and translation.
It exists to answer one question:
If this player were evaluated against professional positional standards, how would trait importance differ?
This section does not replace or override college evaluation.
2. Role Within Player Intelligence
Pro Position Trait Weighting is used to:
translate college player outputs to professional context
stress-test trait profiles against higher standards
support cross-level comparison and normalization
It is not used to:
evaluate college players directly
change college player scores
redefine player truth at the college level
College evaluation remains authoritative.
3. Relationship to College Position Weighting
College Position Trait Weighting:
defines truth for men’s college basketball
drives all college KR outputs
Pro Position Trait Weighting:
exists alongside college weighting
applies only after college evaluation is complete
is used strictly for reference and translation
No blending or averaging occurs between college and pro weightings.
4. Structure
The Pro Position Trait Weighting framework:
uses the same underlying trait definitions
applies different relative importance by position
reflects professional-level role expectations
Trait inclusion and weighting are:
explicit
fixed
position-specific
No dynamic adjustment is permitted.
5. Determinism
Pro Position Trait Weighting is:
deterministic
rule-based
repeatable
Given the same trait scores, the same pro-weighted result is produced.
6. Independence from Context
Pro Position Trait Weighting does not change based on:
team system
league style
role assignment
usage
outcomes
It reflects positional demand, not opportunity.
7. Downstream Use
Pro Position Trait Weighting may be used by:
pro KR rating (reference)
KLVN normalization
long-term projection and translation analysis

It may not be used by:
college team evaluation
recruiting decisions
playing time decisions
scouting outputs for college competition
8. Authority and Change Control
This section is authoritative for:
pro positional weighting structure
relative importance of attributes at the pro level
Any change to:
weights
position definitions
trait inclusion
requires documentation, versioning, and approval.
